# Mobile_Sales_Analysis_Dashboard-PowerBi
“An interactive Mobile Sales Analysis Dashboard built using Power BI, showcasing sales trends, brand performance, customer ratings, and payment insights using Power Query, DAX, and data modeling.”

OVERVIEW:-
This project presents an interactive Mobile Sales Analysis Dashboard built using Microsoft Power BI. The dashboard provides insights into mobile phone sales performance across different cities, brands, months, and payment methods.

The goal of this project is to demonstrate data analysis, visualization, and business intelligence skills using real-world sales data.
